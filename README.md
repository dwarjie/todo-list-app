## Goal

The main goal for this project is to learn:
- Single Responsibility Principle
- Learn modules
- Factory Functions
- Be familiar in using Webpack

## Languages

This project is made using:
- HTML and CSS (Sass)
- Vanilla JavaScript

## Usage

```bash
npm install
npm run build
```
